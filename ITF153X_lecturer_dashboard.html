Dashboard setup
Complete the two steps below to activate the dashboard and automatic student alert emails.
1Anthropic API Key
🔒 Saved only on your browser. Never shared or uploaded.
2EmailJS — Automatic email alerts
How to get these values (free, 5 minutes):
1. Go to emailjs.com and create a free account.
2. Add an Email Service (connect your Gmail or Outlook) → copy the Service ID.
3. Create an Email Template using the variables listed below → copy the Template ID.
4. Go to Account → API Keys → copy your Public Key.

Template variables to include:
{{subject}} · {{lecturer_name}} · {{student_name}} · {{student_no}}
{{flag_reason}} · {{sentiment}} · {{learning_state}}
{{last_message}} · {{message_count}} · {{date_time}}
✅ EmailJS is optional. Leave blank to skip automatic emails — you can always configure it later in the dashboard settings.
